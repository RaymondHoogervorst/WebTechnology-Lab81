Retrieve full data set

URL:
http://localhost:3000
Method:
GET
Response type:
JSON
Description:
Returns a json object containing all products from the database

Add data for new product item

URL:
http://localhost:3000/add
Method:
POST
Request type:
JSON
Description:
Adds the product described in JSON to the database. The properties that need to be defined in JSON are:
  • name
  • origin
  • best_before_date
  • amount
  • image

Retrieve certain product

URL:
http://localhost:3000/item
Method:
GET
Request type:
text
Response type:
JSON
Description:
Returns a json object containing the product with the id given in the request body
